# Dukaan Dosth (దుకాణ్ దోస్త్) 🚀
### *"Aapka dukaan ka dosth, AI ke saath"* | *"మీ దుకాణం దోస్త్, AI తో"*

**Dukaan Dosth** is an AI-powered business assistant designed specifically for small Indian business owners (MSMEs) like kirana stores, street vendors, and small boutiques. It simplifies bookkeeping by allowing owners to log sales, expenses, and credits (udhaar) using **voice or text in Telugu, English, and Tenglish**.

---

## 🛠 Features

- **Voice-First Input**: Tap the microphone and speak your transactions naturally.
- **Multilingual Support**: Fully supports Telugu, English, and mixed-language (Tenglish) inputs.
- **AI-Powered Parsing**: Automatically extracts transaction type, amount, item, and customer details.
- **Daily Business Summary**: Get a warm, conversational end-of-day summary generated by AI in your language.
- **Profit/Loss Tracking**: Visual bar charts and insights to track your weekly business health.
- **Udhaar (Credit) Tracker**: Never forget who owes you money with an integrated credit tracker.

---

## 🏛 Architecture

Dukaan Dosth uses a modern, distributed architecture:

- **Frontend**: React + Vite (Mobile-first, responsive UI)
- **Backend (Auth & Data)**: Spring Boot (Java, PostgreSQL via Supabase)
- **AI Service**: Python FastAPI (Google Gemini 2.0 Flash integration)
- **Database**: PostgreSQL (Supabase)

## 🚀 Getting Started

### 1. Prerequisites
- Node.js 18+
- Java 17
- Python 3.10+
- Google Gemini API Key ([Get it here](https://aistudio.google.com/apikey))

### 2. AI Service Setup
```bash
cd ai-service
python -m venv venv
source venv/bin/activate  # venv\Scripts\activate on Windows
pip install -r requirements.txt
# Copy .env.example to .env and add your GEMINI_API_KEY
uvicorn main:app --reload
```

### 3. Backend Setup
```bash
cd backend
# Add your Supabase DB credentials to application.properties
./mvnw spring-boot:run
```

### 4. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
