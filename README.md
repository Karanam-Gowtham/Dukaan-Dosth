**Dukaan Dosth** is a professional, AI-powered business companion engineered for small Indian enterprises (MSMEs). By merging high-performance **Node.js** architecture with **Google Gemini 2.0 Flash**, it transforms complex bookkeeping into a simple, natural conversation. 

Whether it's a Kirana store, a boutique, or a street vendor, Dukaan Dosth ensures you never lose track of a single Rupee.

---

## ✨ Premium Features

*   🎙️ **Advanced AI Voice Control**: Record sales, expenses, and credits naturally in **Telugu, English, or Tenglish**.
*   📊 **Professional Analytics Dashboard**: A high-density, "Slate & Indigo" enterprise interface providing real-time insights into your shop's health.
*   🧾 **Business Ledger**: A sophisticated, data-dense ledger for auditing all historical transactions with refined status indicators.
*   ✨ **Daily AI Business Summaries**: Receive empathetic, multilingual end-of-day reports generated by Gemini AI to help you plan for tomorrow.
*   📑 **Udhaar (Credit) Management**: A secure, isolated module for tracking customer credits and automated reminders.
*   🔒 **Enterprise Security**: Industry-standard JWT authentication and BCrypt password encryption for your business data.

---

## 🏛 Modern Architecture

Dukaan Dosth leverages a state-of-the-art, distributed stack designed for performance and scale:

*   **Frontend**: React + Vite (Overhauled with a Senior-level Professional Design System, Lucide React, and Framer Motion).
*   **Backend (Core)**: **Node.js (TypeScript)** powered by **Express** and **Prisma ORM** for high-speed, type-safe data access.
*   **AI Service**: Python FastAPI (Google Gemini 2.0 Flash integration for NLP parsing and business insights).
*   **Database**: PostgreSQL via Supabase (Cloud-native performance).

## 🚀 Professional Setup

### 1. Prerequisites
*   **Node.js v24+** (LTS recommended)
*   **Python 3.10+**
*   **Google Gemini API Key** ([Get it here](https://aistudio.google.com/apikey))

### 2. AI Engine (FastAPI)
```bash
cd ai-service
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
# Add your GEMINI_API_KEY to .env
uvicorn main:app --reload --port 8000
```

### 3. Core Backend (Node.js)
```bash
cd backend-node
npm install
# Configure DATABASE_URL (Supabase) and JWT_SECRET in .env
npx prisma generate
npx prisma db push
npm run dev
```

### 4. Professional Frontend (React)
```bash
cd frontend
npm install
npm run dev
```
